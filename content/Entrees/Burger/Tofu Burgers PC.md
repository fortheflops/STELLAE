## Recipe

| Measurements | Ingredients               |
| :----------- | ------------------------- |
| 1 lb         | [[Tofu]], Firm            |
| 1            | Large [[Onion]]           |
| 1            | Large [[Zucchini]]        |
|              | [[Garlic]] (any veg)      |
| 3            | [[Egg Whites]]            |
| 1            | [[Egg Yolk]]              |
|              | [[Tamari]]                |
|              | Fresh Herbs ([[Parsley]]) |
|              | [[Black Pepper]]          |
|              | [[Cumin]]                 |
| ½ Cup        | [[Wheat Germ]]            |
|              | [[Nutritional Yeast]]     |
|              | [[Whole Wheat Flour]]     |
___
### Instructions
1. Sauté Onion.
2. Grate Zucchini.
3. Mix in Whole Wheat Flour, Garlic, Egg Whites, Egg Yolks, Tamari, Fresh Herbs, Black Pepper, Cumin, Wheat Germ, and Nutritional Yeast.
4. "Plop" the mixture, not too thick, on a pan on low heat.

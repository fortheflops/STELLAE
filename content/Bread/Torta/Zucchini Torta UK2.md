## Recipe

| Measurements | Ingredients               |
| :----------- | ------------------------- |
| 2 C                 | Grated [[Zucchini]]     |
| 1 C                 | [[Bisquick]]            |
| ½ C                 | [[Parmesan Cheese]]      |
| 1 C                 | Chopped [[Onion]]       |
| ¼ C                 | [[Oil]]          |
| To taste            | [[Salt]]             |
|To taste            | [[Pepper]]      |
| 1 T                 | [[Parsley]]             |
| As Needed           | Extra [[Parmesan Cheese]]|
|As Needed| [[Paprika]]|
___
### Instructions
1. Mix all ingredients except cheese & paprika. 2. Place in a 9" Pyrex pie plate & sprinkle with cheese & paprika. 3. Bake at 350° for 40 min.

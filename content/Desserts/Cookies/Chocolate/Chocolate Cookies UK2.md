## Recipe

| Measurements | Ingredients               |
| :----------- | ------------------------- |
| 1 C               | [[Sugar]]                   |
| 1 Stick           | [[Oleo]]                    |
| 3 T               | [[Cocoa]]                    |
| 1 t               | [[Vanilla]]                 |
| 1/2 C             | [[Milk]]                    |
| Pinch             | [[Salt]]                    |
| 3 C               | [[Oats]] (minute fine)       |
| 1 C               | [[Nuts]]                    |
___
### Instructions
1. Cook 1 Minute After It Starts Boiling Around Sides.
2. Take Off Fire, Add 3 C Oats (Minute Fine) + 1 C Nuts.
3. Beat & Drop On Wax Paper.
